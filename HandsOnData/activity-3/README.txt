Instructions for Activity-3
===========================

Open SolrDemo project in eclipse. There will be three packages:

1) com.search.mysqlutil 
   - To read data from stackexchange dumps and load it to MySQL

2) com.search.solrj
   - It have example code of indexing and querying using SolrJ in cloud and non-cloud modes

3) com.search.practice
   - Refer SolrJ examples and try them out here.
